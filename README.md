# Unity-GRAD-Playback
Unity project for replaying eye-tracking data and extracting frame-by-frame GRAD values.

This repository contains the custom Unity playback project used to reconstruct and visualize post-experimental gaze behavior relative to a moving virtual character and to extract frame-by-frame Gaze Radial Angular Deviation (GRAD) values.

## Requirements

- Unity 2022.3.20f1
- SRanipal SDK

## Repository structure

- `Assets`: Unity scenes, scripts, and project resources
- `Packages`: Unity package configuration
- `ProjectSettings`: Unity project settings

## Opening the project

1. Download or clone this repository.
2. Open Unity Hub.
3. Select **Add project from disk**.
4. Select the repository folder containing `Assets`, `Packages`, and `ProjectSettings`.
5. Open the project using Unity 2022.3.20f1.

Unity will automatically regenerate the `Library`, `Logs`, `obj`, and other temporary folders.
